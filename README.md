# mcsleep
Code for Multichannel Sleep Spindle Detection using Sparse Low-Rank Optimization

A. Parekh, I. W. Selesnick, R. S. Osorio, A. W. Varga, D. M. Rapoport and I. Ayappa (In review, 2017).
